# 席慕蓉完整时间线

## 一、童年与成长（1943-1953）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1943年10月15日 | 生于重庆城郊金刚坡，祖籍内蒙古察哈尔盟明安旗 | 蒙古血统的起点，战乱年代出生的漂泊命运开端 |
| 1949年 | 随父母迁至香港 | 童年第一次大迁徙，开始体验流离生活 |
| 1953年 | 小学毕业，作文《我的自传》被选登在校刊封底 | 文学天赋初显，首次公开发表作品 |

## 二、台湾求学与绘画启蒙（1954-1963）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1954年 | 迁至台湾，考入北二女（今中山女高）初中二年级 | 在台湾扎根，师从巢静、董秀、杨蒙中等老师 |
| 1954年 | 开始在日记本上写诗，购得第一本诗集余光中《蓝色的羽毛》 | 诗歌创作起步，受余光中影响 |
| 1956年 | 入台北师范艺术科，正式开始习画 | 绘画生涯开端，启蒙老师孙立群、周锳 |
| 1956年 | 以夏采为笔名在校刊发表散文及诗 | 首次使用笔名发表作品 |
| 1959年 | 入师大艺术系 | 专业美术教育深化 |
| 1960年 | 水彩画《静物》入选省美展 | 绘画作品首次入选省级展览 |
| 1963年 | 获全省青年美术比赛水彩第三名，师大毕业美展油画第三名、水彩第二名 | 绘画才华获认可 |
| 1963年 | 以萧瑞为笔名参加皇冠难忘人物征文，《纪念品》获佳作奖 | 文学创作获奖，开始多元笔名创作 |

## 三、欧洲留学与婚姻（1964-1969）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1964年 | 到比利时布鲁塞尔皇家艺术学院进修，入油画高级班，师从里昂·德浮斯教授 | 欧洲艺术深造，视野大开 |
| 1965年 | 作品入选巴黎七十届独立沙龙、八十一届女画家联合沙龙、十届国际妇女绘画展 | 国际画坛崭露头角 |
| 1965年 | 参加比利时皇家历史美术博物馆"中国当代画家展" | 代表中国画家参加国际展览 |
| 1966年 | 在北京举行第一次个人画展 | 首次个展，获最佳优等第一奖、杜特龙·德·特利基金会奖、布鲁塞尔市政府金牌奖、比利时王国金牌奖 |
| 1967年 | 参加比京"欧洲艺展"，获欧洲美协两项铜牌奖 | 欧洲艺坛持续活跃 |
| 1967年 | 进入克劳德·李教授铜版画画室，专习蚀刻铜版画一年 | 版画技艺精进 |
| 1968年 | 在比京艾格蒙画廊及号角画廊同时举行个人画展 | 欧洲画廊双展，艺术生涯高峰 |
| 1960年代 | 在比利时鲁汶大学中国学生中心结识刘海北 | 遇见终身伴侣，为孤独心灵开启光亮之窗 |

## 四、回台任教与创作起步（1969-1980）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1969年 | 以萧瑞为笔名在《中央副刊》发表作品 | 回台后文学活动开始 |
| 1970年 | 以穆伦为笔名在《联合副刊》发表作品；七月回台湾，任教新竹师专美术科 | 回台任教，开始教育生涯 |
| 1970年代 | 以萧瑞、漠蓉、穆伦·席连勃等笔名投稿，作品多为散文 | 多元笔名创作期 |
| 1974年 | 在台湾历史博物馆国家画廊举行回台后第一次个人画展 | 回台后首次个展 |
| 1976年 | 参加联合报第一届小说奖，以千华为笔名，《生日蛋糕》获佳作 | 小说创作尝试 |
| 1977年 | 在台中美国新闻处举行个人画展；10月发表《诗的画，画的诗》专栏 | 诗画结合理念提出 |
| 1979年4月 | 开始研究激光绘画 | 艺术媒介创新 |
| 1979年5月 | 在台北德国文化中心举行个人画展 | 持续艺术创作 |
| 1979年7月 | 《画诗》出版 | 诗画结合作品集问世 |
| 1979年11月 | 在《女性》杂志撰写幼儿美术教育专栏《写给年轻母亲的信》 | 美术教育推广 |
| 1979年12月 | 在台北太极艺廊举行个人画展，为台湾首次激光绘画展出；诗作开始在《联合副刊》发表 | 激光绘画先驱；诗歌正式发表 |
| 1980年 | 继续研究激光绘画，发表论文；7月长诗《我母、我母》在幼狮文艺发表；开始画三百号油画《荷》 | 诗歌创作深化；荷花主题开端 |

## 五、诗歌创作高峰期（1981-1990）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1981年 | 以激光版画参加美国圣地亚哥激光艺术联展；诗作发表于《联合副刊》《中国时报》《中华日报》《幼狮文艺》《台湾时报》《台湾日报》《自由小报》《中华文艺》《阳光小集》 | 诗画双栖，诗歌广泛发表 |
| 1981年4月 | 长诗《爱的名字》在《台湾时报》发表 | 重要长诗发表 |
| **1981年9月** | **诗集《七里香》出版，一个月内再版** | **成名作，台湾诗坛现象级作品，奠定诗坛地位** |
| 1981年10月 | 应邀担任全省美展油画部评审委员 | 艺术界地位认可 |

| 1982年3月 | 散文集《成长的痕迹》及《画出心中的彩虹》出版 | 散文创作丰收 |
| 1982年12月 | 《镭射艺术导论》出版 | 激光艺术理论著作 |
| 1983年2月 | 诗集《无怨的青春》出版 | 第二部重要诗集 |
| 1983年7月 | 与晓风、爱亚合著《三弦》小品文集出版 | 女性作家合作 |
| 1983年10月 | 散文集《有一首歌》出版 | 散文代表作 |

| 1988年3月 | 诗及散文合集《在那遥远的地方》出版 | 乡愁主题作品 |
| 1988年7月 | 赴印尼里岛作荷花写生 | 荷花主题持续 |

| 1989年3月 | 散文及素描集《写生者》出版 | 写生主题作品 |

---

*注：席慕蓉，原名穆伦·席连勃（蒙古语意为大江河），笔名萧瑞、漠蓉、穆伦、千华、夏采等。1943年生于重庆，祖籍内蒙古察哈尔盟明安旗。台湾师范大学艺术系毕业，比利时布鲁塞尔皇家艺术学院油画高级班进修。曾任新竹师专、东海大学美术系教职。1989年首次回蒙古高原寻根，成为创作重要转折点。其诗歌以爱情、乡愁、时光、生命为主题，影响了几代华人读者。*
