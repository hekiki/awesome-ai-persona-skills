# 琼瑶完整时间线

## 一、童年与成长（1938-1954）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1938年4月20日 | 生于四川成都，祖籍湖南衡阳 | 战乱年代出生的漂泊命运开端 |
| 1942年 | 随父母迁居湖南衡阳 | 童年在战乱中度过 |
| 1949年 | 随父母赴台湾，定居台北 | 人生重大转折，在台湾扎根 |
| 1954年 | 考入台北第二女中 | 开始文学创作，在校刊发表作品 |

## 二、高中与初恋（1954-1957）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1954-1957年 | 就读于台北第二女中 | 文学天赋初显 |
| 1957年 | 与国文老师产生师生恋 | 《窗外》原型，影响一生的情感经历 |
| 1957年 | 高考落榜 | 人生挫折，成为创作素材 |

## 三、第一次婚姻（1959-1964）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1959年 | 与庆筠结婚 | 追求爱情，但婚姻不幸 |
| 1961年 | 生子陈中维 | 成为母亲 |
| 1963年 | 开始写作《窗外》 | 将痛苦经历转化为创作 |
| 1964年 | 与庆筠离婚 | 婚姻失败，但创作起步 |

## 四、成名与《窗外》（1963-1965）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1963年 | 《窗外》出版，一举成名 | 开启作家生涯 |
| 1963年 | 与平鑫涛相识 | 遇见终身伴侣和事业伙伴 |
| 1964年 | 《几度夕阳红》出版 | 奠定言情小说地位 |
| 1964年 | 《烟雨濛濛》出版 | 代表作之一 |

## 五、创作高峰期（1965-1980）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1965年 | 《六个梦》系列开始出版 | 创作进入高峰期 |
| 1970年代 | 陆续出版《在水一方》《我是一片云》《月朦胧鸟朦胧》等 | 成为台湾最畅销作家 |
| 1974年 | 《一帘幽梦》出版 | 代表作之一 |
| 1979年 | 与平鑫涛结婚 | 16年等待终成正果 |

## 六、琼瑶剧时代（1980-2000）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1985年 | 《几度夕阳红》改编电视剧 | 开启琼瑶剧时代 |
| 1986年 | 《烟雨濛濛》改编电视剧 | 刘雪华成为第一代琼女郎 |
| 1987年 | 《庭院深深》改编电视剧 | 再创收视佳绩 |
| 1990年 | 《六个梦》系列改编电视剧 | 《婉君》《哑妻》等 |
| 1990年 | 《雪珂》改编电视剧 | 刘雪华主演 |
| 1992年 | 《青青河边草》改编电视剧 | 金铭成为童星 |
| 1993年 | 《梅花三弄》系列 | 《梅花烙》《鬼丈夫》《水云间》 |
| 1994年 | 《两个永恒》系列 | 《新月格格》《烟锁重楼》 |
| 1996年 | 《一帘幽梦》改编电视剧 | 陈德容、萧蔷主演 |
| **1998年** | **《还珠格格》播出** | **创造47%平均收视率神话** |
| 1999年 | 《还珠格格第二部》播出 | 延续收视奇迹 |
| 2001年 | 《情深深雨濛濛》播出 | 赵薇、林心如主演 |
| 2003年 | 《还珠格格第三部》播出 | 黄奕、马伊琍主演 |

## 七、晚年与淡出（2000-2019）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2007年 | 《又见一帘幽梦》播出 | 新版改编 |
| 2011年 | 《新还珠格格》播出 | 翻拍经典 |
| 2013年 | 《花非花雾非雾》播出 | 最后一部电视剧作品 |
| 2014年 | 起诉于正抄袭案 | 知识产权保护标志性案件 |
| 2015年 | 法院判决琼瑶胜诉 | 维护原创权益 |
| 2017年 | 出版《琼瑶作品光影辑》 | 回顾创作生涯 |
| 2018年 | 与平鑫涛子女发生安乐死争议 | 家庭矛盾公开化 |
| **2019年5月** | **平鑫涛去世** | **挚爱离世，创作生涯终结** |

## 八、最近动态（2020-2026）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2020年 | 淡出公众视野 | 享受晚年生活 |
| 2021年 | 偶尔在社交媒体发声 | 与读者保持联系 |
| 2022-2026年 | 基本不再公开露面 | 正式告别创作生涯 |

## 九、创作生涯统计

| 项目 | 数据 |
|------|------|
| 小说作品 | 60余部 |
| 电视剧改编 | 50余部 |
| 创作年限 | 1963-2013（50年） |

## 十、关键转折点

| 时间 | 转折点 | 意义 |
|------|--------|------|
| 1963年 | 《窗外》出版 | 从学生到作家 |
| 1964年 | 第一次婚姻失败 | 痛苦转化为创作动力 |
| 1979年 | 与平鑫涛结婚 | 事业与爱情双丰收 |
| 1985年 | 首部电视剧改编 | 开启琼瑶剧时代 |
| 1998年 | 《还珠格格》播出 | 创造收视神话 |
| 2014年 | 起诉于正 | 知识产权保护 |
| 2019年 | 平鑫涛去世 | 创作生涯终结 |
