# 天下霸唱（张牧野）完整时间线

> 来源：百度百科、公开采访资料整理

---

## 一、早年经历（1978-2005）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1978年 | 出生于沈阳，本名张牧野 | 人生起点，后成为"盗墓小说鼻祖" |
| 1978-1980年代 | 随父母在内蒙古赤峰和东北大兴安岭一带生活 | 父母为物探队队员，长期在野外寻找矿脉，童年接触大量地质、风水、民间传说知识 |
| 童年时期 | 住在天津胡同里，挖死人骨、听当地人讲鬼故事 | 这些经历成为日后创作《鬼吹灯》的素材源泉 |
| 小学时期 | 就读于内蒙古赤峰山里的地质队机关学校 | 两年后回到天津 |
| 中学时期 | 作文成绩不太好，汉字只认识两三千个，但喜欢看《隋唐演义》《水浒传》等古典文学 | 写作风格受古典文学影响，爱听民间传说、奇闻逸事 |
| 高二 | 因数学从未及格，选择辍学打工 | 人生转折点，放弃传统学业道路 |
| 辍学后 | 在天津洗盘子，后到深圳合资企业做打杂工 | 底层打工经历，积累社会阅历 |
| 打工期间 | 通过自学获得专升本美工专业文凭 | 自我提升，为后续工作铺路 |
| 之后 | 去电视台做美工，但因觉得没学到实际东西而辞职 | 认识到自己不适合传统工作模式 |
| 创业初期 | 从事服装生意、经营美容院等多种生意 | 多元化创业尝试 |
| 约2000年代初 | 和一帮朋友回天津开了家金融公司 | 进入金融行业，担任投资经理 |
| 金融公司期间 | 在山西帮人办理开矿批文，每天下矿，陪煤老板喝酒 | 接触风水先生，听他们讲述风水知识和江湖经历，为创作积累素材 |

## 二、创作起步（2005-2006）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2005年 | 27岁，创业公司遭遇低谷，朋友痴迷天涯论坛《莲蓬鬼话》板块 | 人生低谷期，为后续创作埋下伏笔 |
| 2005年 | 编了《凶宅猛鬼》故事发在天涯论坛，迅速获得网友热爱 | 首次网络创作尝试，获得成功 |
| 2005年 | 接着编了《雨夜谈鬼事》和《阴森一夏》（后改写为《迷航昆仑墟》出版） | 写作一发不可收拾，奠定网络作家基础 |
| 2005年 | 与起点中文网签订合作协议，正式开始写手之旅 | 进入专业网络文学创作领域 |
| 2005-2006年 | 每天需要完成3000字写作任务，曾有一次写了1万多字 | 创下写作生涯最高纪录，展现惊人创作力 |
| 2006年3月 | 在军事论坛上发表《鬼吹灯》故事 | 《鬼吹灯》诞生，开创中国盗墓小说流派 |
| 2006年 | 小说取名《鬼吹灯》，源自东北俗语"烟泡鬼吹灯" | 八本《鬼吹灯》，每写完一本都要醉一次 |
| 2006年12月 | 《鬼吹灯》上榜新浪读书风云榜 | 不到一年时间风靡网络，被尊为"中国盗墓小说鼻祖" |

## 三、出版与版权（2007-2015）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2007年 | 好莱坞找到天下霸唱，想把《鬼吹灯》拍成美国版（情节人物全换美国） | 遭到拒绝，体现对作品的珍视 |
| 2010年9月下旬 | 公开表示推出大结局《牧野诡事》，决定通过公开招标确定出版方 | 开始重视IP管理和运营 |
| 2010年10月 | 以总价1000万元签约北京新华先锋 | IP管理和运营情况得到很大改善，进入商业化新阶段 |
| 2013年3月10日 | 携新书《河神·鬼水怪谈》在南京新街口新华书店签售 | 作品多元化发展 |
| 2015年 | 着手筹备小说《大耍儿》，采访了许多当年混迹江湖的老大哥 | 回归天津本土题材创作 |
| 2015年5月 | 好友孔德胜引荐向上影业肖飞，起初谈《死亡循环》项目开发 | 进入影视改编领域 |
| 2015年6月 | 成立向上霸唱工作室，为向上影业子公司，独立运营作品 | 向上影业有优先购买权 |
| 2015年下半年 | 以手中IP入股，担任向上影业首席内容官 | 从作家转型为内容产业经营者 |

## 四、影视改编与版权纠纷（2015-2019）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2015年9月30日 | 《鬼吹灯》改编第一部电影《九层妖塔》上映 | 开启《鬼吹灯》影视化时代 |
| 2015年12月18日 | 《鬼吹灯》改编第二部电影《寻龙诀》上映 | 天下霸唱担任编剧顾问，该片成为IMAX国产片票房冠军 |
| 2016年6月 | 天意影视与派乐传媒签订《鬼吹灯之摸金符》剧本改编项目书 | 与张永琛联合担任总编剧 |
| 2016年9月 | 授权的第一款游戏《鬼吹灯摸金校尉》页游上线 | 游戏领域拓展 |
| 2017年 | 将3万字短篇改编成18万字长篇小说《天坑鹰猎》 | 小说完成前就开始电视剧剧本创作，担任剧本总监 |
| 2017年7月19日 | 《河神·鬼水怪谈》改编网剧《河神》在爱奇艺上映 | 豆瓣评分高达8.1，口碑爆款 |
| 2017年8月23日 | 《天坑鹰猎》新书发布会在北京举行 | 与王俊凯等明星合作，扩大影响力 |
| 2017年9月29-30日 | 舞台剧《摸金玦》在上海大宁剧院亮相 | 首部舞台剧作品，书未出版电影改编权已拍出4000万元 |
| 2018年1月 | 《河神》出版，为创作专程前往重庆磁器口采风 | 注重实地考察的创作态度 |
| 2019年 | 创作完成《天坑宝藏》，作为《天坑鹰猎》续篇 | 系列化创作持续进行 |

## 六、重要版权纠纷

| 时间 | 事件 | 意义 |
|------|------|------|
| 2007年前后 | 《鬼吹灯》版权以10万元价格出售给起点中文网（阅文集团前身） | 早期对IP价值认知不足，导致后续版权纠纷根源 |
| 2015-2019年 | 因早期版权出售，天下霸唱无法使用"鬼吹灯"标题创作新书 | 版权意识觉醒，成为作家IP保护的典型案例 |
| 2019年前后 | 天下霸唱诉《九层妖塔》侵犯著作权案 | 维护自身权益，引发行业对改编权边界的讨论 |
| 持续至今 | 因早期版权合同限制，后续《鬼吹灯》系列影视改编与原作者关系复杂 | 成为中国网络文学IP版权纠纷的标志性案例 |

## 七、创作特点总结

1. **地质背景影响**：父母物探队工作，童年接触风水、地理、民间传说
2. **天津本土文化**：胡同生活、江湖文化、方言俚语融入作品
3. **实地考察**：为写《河神》专程去重庆磁器口采风
4. **高产创作**：曾日更万字，保持持续产出
5. **IP意识觉醒**：从早期10万卖版权到后来千万级签约，再到成立工作室
