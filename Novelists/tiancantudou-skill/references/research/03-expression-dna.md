# 03 — 表达DNA与风格

## 核心语言DNA：「退婚羞辱→金手指觉醒→打脸升级」循环

### 1. "莫欺少年穷"叙事弧线
这是天蚕土豆最标志性的叙事DNA：
- 开局羞辱（退婚/废物/被看不起）→ 情绪压到最低
- 金手指觉醒（药老/异火/焚诀）→ 希望的转折
- 三年之约（明确时间节点）→ 读者期待拉满
- 打脸复仇（云岚宗一战）→ 爽感爆发
- 新地图→新羞辱→新升级→新打脸（循环）

### 2. 等级体系即叙事引擎
斗气等级本身就是叙事的"骨架"：
斗之气→斗者→斗师→大斗师→斗灵→斗王→斗皇→斗宗→斗尊→斗圣→斗帝
每一个等级的突破都是一次叙事高潮。"突破"这个动作是天蚕土豆最核心的爽点制造机。

### 3. 战斗描写公式
```
[敌人]轻蔑一笑："不过如此"
[主角]嘴角上扬，露出一抹冷笑
[异火]猛然爆发，[技能名]
[敌人]脸色骤变："这不可能！"
[围观者]倒吸一口凉气："恐怖如斯！"
```

"恐怖如斯"是天蚕土豆的标志性用语——几乎每场战斗都会出现。

### 4. 地图跃迁结构
乌坦城（小城）→ 云岚宗（宗门）→ 中州（大陆）→ 大千世界（宇宙）
每换一个地图，主角从"底层"重新开始，然后一路升级到"顶层"。这是"换地图重启"的经典模板。

### 5. 爽点节奏控制
- 每50-100章一个小高潮（突破/打脸/收服）
- 每200章一个大高潮（换地图/复仇完成）
- 压抑与释放交替：先压（受辱/遇险）→ 再弹（爆发/反转）
- "三十年河东，三十年河西"的情绪锚点反复出现

### 6. 配角设计法则
- 随身老爷爷/导师（药老类型）——提供知识和资源
- 美女盟友——每个地图一个，功能性角色
- 富二代反派——看不起主角，被反复打脸
- 势均力敌的对手——最终成为盟友

### 7. 从后宫到专一的进化
- 《斗破苍穹》：萧薰儿+美杜莎+云韵+小医仙（多女主暧昧）
- 《武动乾坤》：应欢欢+绫清竹
- 《大主宰》起：仅洛璃一人（彻底单女主）
- 《万相之王》：姜青娥一人，势均力敌姐弟恋
