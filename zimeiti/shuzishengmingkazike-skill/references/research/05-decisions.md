# 05 — 创立逻辑与关键决策

## 1. 前职业：交互设计师 → 设计总监 → AI产品探索者
- UI/交互设计师出身 → 设计总监（公募基金数据分析平台创业公司）
- 22年底ChatGPT上线→ 全公司用AI
- 23年9月第二批算法备案（金融领域第一个）
- 23-24年尝试做Tob AI产品（卖给基金公司和券商），客户推广不顺
- 24年3月公司业务调整→离职→"变身肝神卡兹克"
- **核心判断**：设计总监经历带来产品思维+技术审美+对"什么能直接用"的直觉

## 2. 2023年2月——公众号第一篇文章
触发器：《流浪地球》的热爱 → 用"数字生命"命名（源于电影设定）
早期：传统创作方式极重——一个4秒3D建模视频镜头需要一周
ChatGPT+AI绘图兴起→发现极大契合自己需求（"尤其是对于像我这样的懒人"）
策略：将探索AI工具的过程视为一种游戏
结果：开始研究AIGC，拓展内容创新

## 3. 2023年5月——《流浪地球3》AI预告片出圈
策略：量大管饱——"AI根本不会按预期效果来，生成几百个镜头，总能拼出满意的"
数据：693张MJ图 + 185个Gen2镜头 → 精选60个
结果：全网千万播放，上CCTV6，郭帆导演团队主动联系→ 正式成为AI内容创作矩阵

## 4. 2024年3月——全职运营"数字生命卡兹克"
触发器："公司业务调整，离职" → 出走→all in AI自媒体
自称"肝神卡兹克"——高强度持续产出
策略：用AI自动化实现高频产出
"每天搜集全网AI资讯，AI打分筛选→10条→30多个社群早报"
"用AI分析社群讨论焦点→帮助产生选题"

## 5. 内容方法论大公开——10条方法论
"活人感是AI时代最贵的奢侈品"作为第一基石
精髓在于方法论的结构化体系化，超越技巧汇集
持续输出——"2年换了2个人"表明其内容策略持续迭代

## 6. 三周年大字碑反思（2026年2月）
9条心得环环相扣：花钱用最好的→每周自动化一个任务→实习生思维→AI能帮我吗→创造点什么→警惕幻觉→不等准备好→培养品味→把时间还给现实中的人
核心立场："我们还处于AI早期，0.04%的人才真正在用AI工具"
但"还来得及"——面向大部分普通人启蒙

## 7. 2026年4月——万人级AIFUT大会
 motivations主要有三点:
- "属于普通人的AI嘉年华"
- 私心：需要一个舞台，一个正当理由邀请想见的大佬
- 倾尽资源：人脉+金钱+时间
结果：1500人现场+200万线上观看+34家企业展区+5V5电竞赛
首次将AI内容输出从"个人自媒体"升级为"平台级行业活动"
