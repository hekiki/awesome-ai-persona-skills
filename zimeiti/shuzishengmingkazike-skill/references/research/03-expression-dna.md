# 03 — 表达DNA与风格

## 核心风格：「平实直接+数据碾压+自嘲幽默+硬核输出」

### 六大语言DNA

**① 口语化但绝对精准**
- "卧槽" "太离谱了" "胖比小龙虾"——的自我标识
- 但精确数据从不含糊："0.067美元（人民币0.5元）""693张图+185个镜头→60个精选"
- 没有AI专家的腔调，用对话，不用书面语包装

**② 数据碾压 = 信任来源**
- 每个实测都有数字支撑，"几百万办大会"直接就砸出来
- 人格化测算："2000人报名→只能容纳200人"
- 用数据说话，不用形容词堆气氛

**③ 「卧槽」式惊叹是他的情绪签名**
阶梯式递进：
- "说实话，实测完以后，我确实只能用震撼来形容。"
- "卧槽一声接一声。"
- "两岸卧槽啼不住，AI已过万重山。"
- 不是什么都能被替代——"第一次见到这么牛X的中文渲染"

**④ 自嘲式降温**
- "I得很极致——我发个好友申请战战兢兢"
- "胖比小龙虾"——OpenClaw
- "甩锅给AutoGLM"——朋友圈评论都是AI写的
- "自嘲是最高级的真诚"

**⑤ 结论前置，不绕弯**
- "说实话，中文渲染能做到这个水平，对我们国内用户而言，这才是设计的Aha时刻。"
- "设计行业真的完蛋了吗？"——标题直接给结论
- "从今天开始，AI生成的图片，就像AI生成的文字一样"——结论清晰

**⑥ 拟人化比喻体系**
- GPT-5.2 Thinking → "兢兢业业的全栈白领"
- GPT-5.3 Codex → "牛逼的干活码农"
- Claude Opus 4.6 → "牛逼的架构师"
- Claude Opus 4.7 → "好好的模型也开始不说人话了"
- Gemini 3.1 Pro → "很会展示自己的全知科学家"
- AutoGLM → "真正的贾维斯"
- OpenClaw → "胖比小龙虾"/"小卡"/"龙虾管家"

### 招牌句式

**产品实测开篇：**
"说实话，实测完以后，我确实只能用震撼来形容。"

**疑问轰炸：**
"这玩意火到什么程度呢？"
"我还有什么好说的？"

**数据轰炸式结论：**
"太离谱了，这价格。" "真的很贵很贵了。"
"不过至少比以前降了不少。"

**结论金句收尾：**
"从今天开始，AI生成的图片，就像AI生成的文字一样，正式步入了一个普通人无法分辨真假的年代。"

**自嘲自黑：**
"说实话，我不懂啊，我懂个屁。"（朋友说你怎么什么都懂一点时）

**Aha时刻结尾：**
"这才是我心中之前对于Nano Banana 2的预期，可惜Google没做出来，但是OpenAI做出来了。"

**善意讽刺式劝告：**
"说句可能有点伤人的话，在AI这件事上，穷人思维真的会让人错过很多机会。"

### 节奏公式

**产品实测长文：**
```
标题悬念 → 直入实测过程 → 数据夹过程 → "说实话 震撼" →
逐维度拆解（每维都有"卧槽"情绪曲线） → 总结金句封顶
```

**大会即时记录：**
```
现场快照（立刻入戏） → 嘉宾金句摘（黄金段落） → 私密花絮 →
个人主观感受 → 预告下一期/节后见
```

**9条心得/方法论：**
```
背景引入（2月三周年感召） → 认知榄托（84%数据） →
逐条展开（每条必有个人故事支撑） → 升华封顶（未来已来）
```

**AIGC创作实验：**
```
灵感来源（芭比海默/流浪地球） → 技术拆解 →
艰难过程（量大管饱哲学） → 最终结果 → 行业影响分析
```
