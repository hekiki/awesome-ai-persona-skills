## 核心作品详解

### 汉语新解（全网爆火）
- 用Lisp伪代码编写，适用Claude Sonnet
- 输入任意汉语词汇，自动生成一针见血的精辟解释卡片
- 风格：讽刺+幽默+精准，卡片复古美学呈现
- 衍生上千个版本：周易占卜/经济学新解/职场新解/各学科新解
- 李继刚本人说这是他**最不喜欢的作品**，因为「偏闲暇娱乐」

### 七把武器系列
| 武器 | 功能 |
|------|------|
| 哲学家 | 多角度深度理解一个概念 |
| 逻辑之刃 | 逻辑分析与推理 |
| 类比之弓 | 用类比帮助理解 |
| 多棱镜 | 多维度审视 |

### ljg-skills（13+1认知工作流）
- ljg-plain（逼AI说人话）
- ljg-learn（概念解剖器）
- ljg-word（单词精通）

- 底层逻辑：**精准压缩**——把复杂的东西压进高密度核心，让AI自己展开细节

## 写作风格特征

### 标题风格
- 哲学追问式：`当我们说Prompt时，到底在说什么？`
- 极简有力：`公文笔杆子`、`汉语新解`、`逻辑之刃`
- 不用数字噱头、不制造焦虑、不标题党

### 内容结构
1. **从困惑入手**：先描述大家普遍遇到的困境
2. **哲学追问**：不停留在工具层面，追问本质
3. **概念拆解**：用Lisp伪代码或结构化方式呈现
4. **留白思考**：不给标准答案，让读者自己领悟

### 语言特征
- Lisp伪代码风格：`(defun 哲学家(用户输入)...)`
- 中英文混用但克制
- 版本迭代标注规范：`作者:李继刚;版本:0.7;模型:Claude Sonnet;用途:...`
- 极简主义，不说废话
