# 03-expression-dna —— 特工宇宙表达DNA

---

## 整体文风标签

| 维度 | 描述 |
|------|------|
| 调性 | 工科生体查+年轻人轻盈感+产品测评+实操导向，无距离感 |
| 语言 | 纯中文，口语化，喜欢用比喻和梗来降低技术门槛 |
| 叙述视角 | 第一人称「我们」+「特工们」，「特工宇宙」品牌自我锚定频繁出现 |
| 节奏 | 短段落+快节奏+干货密集型，像在看一个有实际经验的产品经理在给你讲 |

---

## 句式特征

### 1. 踩坑导向 + 「沉浸式体验」，非被动搬运

> 「我们花了10万块，只为这一份AI PPT口碑榜……经过百名观猹员们的深度体验……」

**区别于**其他AI自媒体的「介绍一下XX产品是什么」——特工宇宙的文章总是「我们先试过，然后告诉你什么情况好用、什么不行」。

### 2. 「特工」品牌世界构建

特工宇宙把自己构建成一个宇宙：
- 读者是「特工人」
- 加入社群要「任务」「加特」
- 工作内容就叫「任务」：「发布了一份XX任务简报」
- 组织内部用代号：特工少女、特工小鹏、特工小氧、特工炸炸

### 3. 过年式热点追击

> 「今年过年期间，随着DeepSeek横空出世，以及Manus的爆火，特工宇宙连续推出十余篇深度研究推文。」

出现一个热点，特工宇宙的反应模式是：**连续多篇、高密度追击**，一篇追一篇，不是蹭一下就走。

### 4. 实操干货直接交付

> 「10分钟，搭建一个智能体」「发布到微信/飞书/豆瓣等平台」

每篇文章结尾几乎都有实操步骤或可执行建议。不空谈趋势，给具体做法。

### 5. 「不是搬运工」的自我定位

特工宇宙经常强调「我们不只是AI搬运工」：
- 不只把别人的论文翻译一遍
- 不只把英文内容搬运到中文
- 要说「为什么重要」「意味着什么」「你应该怎么做」

## 标题特征

1. **「XX 深度解析/全解读」**：「OpenManus 深度解析」「AIAgent 法案 全面解读」
2. **「XX 实测/实测报告」**：「我们花了10万块，只为这一份AI PPT口碑榜」
3. **「XX 怎么用/如何做」**：「DeepSeek+扣子：10分钟搭建一个智能体」
4. **「XX 年/阶段 报告/榜单」**：「2024年度AI应用榜单」，「AI Agent 圣经」
5. **「XX 火了/炸了/走出圈」**：「Manus 出了圈」，，「DeepSeek 炸了」

## 写作气质与技巧

### 1. 「我们亲自试过」的信息验明

特工宇宙的文章总是出现「我们测了」「我们体验了」——这是核心段位认证。

### 2. 「直观反馈 + 详细解读」的双层结构

- B层：先告诉你直观体验「我觉得XX挺好用的，但是…」
- 上层：给你深度解读「为什么会这样，背后的技术逻辑是…」

### 3. Humor & 梗

- 不追求搞笑，但偶尔用梗让技术文章更接近生活
- 对话中常有一种「和稀泥类比」，来解释技术问题

### 4. 「超年轻特工」

- 创业8个月后就能拿到 谷歌海外大厂邀请参会的殊荣
- 青少年创业 / 00后CEO tag 经常作为「含量潮」气质
- 这种「小小年纪大成就」的叙述在该行业话语结构里特别跑火
