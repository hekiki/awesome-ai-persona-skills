# 04-external-views —— 特工宇宙外部视角与行业内评价

---

## 行业地位

特工宇宙在AI自媒体圈属于**垂直赛道+年轻力量**的独特存在。

- 国内首个专注 AI Agent 的垂直自媒体
- 0-1创业，在不到2年时间吸引到 红杉中国 + 华兴资本联合投资
- 高密度、高品质粉丝（15万，60%互联网从业者，大量投资人/创业者/学者订阅）
- 谷歌等海外大厂邀请参与研讨活动的「圈层渗透力」

### 与同行 AI 科技媒体的定位差异

| 账号 | 核心差异 | 和特工宇宙的关系 |
|------|---------|----------------|
| 36氪 | 泛科技商业媒体, AI 部分偏时效 | 合作伙伴（36氪圆桌同台） |
| 硅星人 | 硅谷一线视角+中美对比 | 竞品但重叠度低（聚焦Agent vs 聚焦硅谷AI全景） |
| 机器之心 | 论文+工程深度 | 竞品，偏学术 vs 偏实操产品 |
| 量子位 | 短讯包+月度总结 | 竞品，信息密度 vs 垂直深度 |
| 新智元 | AI技术长文偏科研 | 竞品，偏研究与偏用户体验 |
| 屠龙之术（庄明浩） | AI产品方法论 | 合作伙伴/战略顾问 |

### 独特优势

1. 专注 AI Agent 这个**最热的垂直赛道**，诠释信息壁垒
2. **年轻合伙人**（00后）与垂直人群建立新鲜的身份标签
3. **实际 Agent 开发能力**认证场阅，不纯讨论者
4. **「观猹」平台**补齐自媒体→社区→产品的路径
