# 05-decisions —— 特工宇宙关键决策与内容策略选择

---

## 1. 2024年初创立，专注 AI Agent 单一垂直赛道

**背景**：2023底-2024初大模型全面爆发，媒体圈突然涌入大量内容。

**决策逻辑**：
- 不做泛AI媒体，专精 AI Agent 垂直赛道——「必须非常非常垂直深耕」
- 怼赢 AI Agent 这一条赛道，成为圈内最有话语权的号
- 逻辑异常简单：大厂+自媒体几乎都在讲大模型，但很少有人真正讲懂 Agent

**效果**：2024春节「连续推出10+篇深度推文」——在还没有任何「特工宇宙」品牌认知的情况下，靠内容质量杀出一条血路

---

## 2. 双轨制：媒体+Agent开发并行

**背景**：团队不只是写，也自己开发 Agent 产品。

**决策逻辑**：
- 「只写不写」和「只用不写」之间的张力，让内容质量感知「现实的体感」
- 与支付宝合作开发 Emoji翻译器/黑话翻译器等实际 Agent——本身就是最好的案例库
- 做 toB Agent 项目让自己的商业路径更独立于纯媒体

**效果**：拥有比纯媒体账号**更大的原型注射**能力和对产业格局理解

---

## 3. 「观猹」平台：从内容粉丝社区到AI产品分发平台

**背景**：越做AI Agent，越发现「社区=产品分发+用户反馈+口碑磁场」

**决策逻辑**：
- 只靠公众号不够，需要建立自己的内容社区
- 「观猹」定位：「AI应用的大众点评」，让用户像刷小红书一样刷AI产品测评
- 卡片式陈列+一键试玩+打分评价，是精准对标 Product Hunt 的本土化改造

**效果**：AI PPT 口碑榜（10万元成本+百名观猹员实测）在圈内建立了「真测过」的品牌认可

---

## 4. 体验 1200+ Agent 的个人知识积累

**背景**：仲泰和团队成员在校期间就是「玩 Agent 的跟品员」，不是AI圈的半路出家人。

**决策逻辑**：
- 优先「体验量」：「我体验过各领域1200多个Agent」——资历felt式积累
- 实战演练带货「深度研究推文」的信源底气

**效果**：真正写推文、评测内容时，是「1100+案例经验」的产物，而非信息搬运

---

## 5. 把测评做成本，信息产品化

**背景**：AI PPT口碑榜花了「10万元」成本。

**决策逻辑**：
- 让「观猹员」亲自测——把钱花在用户，换取真实体验数据
- 把钱不仅仅视为流量成本，而是产品信任成本

---

## 决策模式总结

1. **「垂直单点先突破」**：先做一个赛道做到极致——AI Agent，而不是同时追多个热点
2. **「读者（用户）能感知到实际差异」**——不是靠「我知道」而是「我试过」来积累信任
3. **「内容即获客，产品即留存」**：公众号涨粉→观猹平台留存→Agent开发商业化，三重路径并行
4. **「不搬运工」态度**：不满足于翻译/搬运，要有独立判断和创新输出
