# 03 — 表达DNA与风格

## 核心风格：Why追问 + 历史纵深 + 创业者伙伴视角

### 五大语言DNA

**1. Why追问型叙事——"他为什么要做这件事？"**
- 不是报道事实，而是追问动机
- 「真正成功的创新，往往源于一个独特的Why」
- 「Why的背后是一个人过去的积累、独特的认知凝结」
- 标题风格：不是「XX发布了」，而是「XX凭什么让YY疯狂上头」

**2. 历史纵深叙事——"20年后的回望"**
- 善于用历史案例论证当下趋势
- 张一鸣2014硅谷行 → 2017收购musical.ly → TikTok全球化
- 2014马斯克来华 → 智能电动车非共识 → 今天的电动车大战
- 2015库兹韦尔预言 → 2016AlphaGo → 今天的AGI浪潮
- 这种写法让文章有"预言验证"的厚重感

**3. 创业者伙伴视角——"不做评判，做陪伴"**
- 不居高临下地评价创业者
- 而是以"伙伴"身份记录和思考
- "护送大家走一段夜路，就是社区存在的意义"
- 内容不是终点，是创业者行动的起点

**4. "见识-认知-行动"三段论结构**
- 每篇文章/演讲都隐含这个结构
- 先讲见识（发生了什么）
- 再讲认知（这意味着什么）
- 最后讲行动（你该怎么做）
- 这不是显性框架，而是底层叙事DNA

**5. 内容原教旨主义——"长文+深度"**
- 不追5分钟快讯的时效性
- 17万字年度特刊——极致深度
- 创始人对谈实录完整发布——不做删减
- 演讲全文逐字发布——不摘要不缩写
- 对速度妥协，对深度执着

### 招牌句式

**Why追问式开篇：**
```
大家都在讨论XX做了什么（What）、怎么做（How）。
但我觉得最根本的，其实是——他为什么要做（Why）？
```

**历史纵深式论证：**
```
这让我想起2014年……
当时还……但今天……
```

**创业者伙伴式收尾：**
```
如果你决定要创业/亲自登山了……
我们会……做一直陪你前进的……
```

### 节奏公式

**深度对谈节奏：**
```
[嘉宾介绍/背景铺垫] → [Why追问] → [历史案例/类比] → [当下判断] → [行动建议]
```

**年度演讲节奏：**
```
[个人故事开头] → [极客公园是谁] → [为什么存在] → [能为你做什么] → [Why not呼吁]
```

**趋势特稿节奏：**
```
[现象引入] → [数据/案例支撑] → [历史脉络回溯] → [未来预判] → [创业者启示]
```
