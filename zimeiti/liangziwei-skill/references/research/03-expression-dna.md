# 03 — 表达DNA与风格

## 核心风格：「数据驱动·短快讯·深报道·团队出品」

### 六大语言DNA

**1. 短新闻快讯体——「新闻5要素式极速输出」**
- 每篇100-300字报道一条新闻事件
- 严格遵循「标题/一句话信息/关键数据」结构
- 标准品牌：量子位 | 公众号 QbitAI
- 日更50+条，信息密度极高
- 《标题》用书名号，正文不矫饰

**2. 数据驱动的客观报道**
- 关键数据必须列出：具体的数字支撑
- 「4.55亿美金！中国具身智能最大单笔融资诞生」
- 「年薪突破百万大关」——招聘文里的数据冲击
- 整体立场：数据导向、不引申、不预设立场

**3. 团队署名制——「编辑部 发自 凹非寺」**
- 这是量子位独有的版式花押：编辑部+发自凹非寺+量子位|公众号QbitAI
- 每篇文章均有署名作者，体现团队协作感
- 不出现个人IP感——量子位本身是品牌不是个人

**4. 报道+评论融合体**
- 纯新闻：「刚！XX重磅发布！」
- 加观察：「这意味着XX格局将发生改变」或「XX是否会改写行业」
- 不完全中立但也不站队：量化等于「这意味着」
- 保持观察者视角：定性观察 + 当事人采访

**5. 标题风格——「惊叹号+关键词」**
- 惊叹号密集：「刚！年仅！刷屏！杀疯了！」——情绪密度极高
- 关键词前置：科技关键词+商业关键词
- AI行业事件一般在第一线：第一时间快速走天下

**6. 叙事节奏——「新闻体 + 故事体 + 报告体」三合一**
- 新闻快讯：电报体
- 深度报道：叙事故事性+行业分析
- 研究年报：矩阵+数据+方法论+展望

### 招牌句式

**新闻快讯极速体：**
```
刚！[事件]！
[一句话描述] [核心数据]
[跟进/下一步/意义]
——量子位 | 公众号 QbitAI
```

**年度报告系列体（研究报告格式）：**
```
量子位智库发布《XXXX》
[一段摘要性介绍]
随后是趋势/判断/数据矩阵
[正文展开]
[相关阅读]
```

**深度特稿开篇：**
```
[悬念标题]
[感叹/数据/现象支撑] + 事件叙述
```

### 节奏公式

**新闻快讯（主流格式）：**
```
Title → 一句话信息（主体+数据） → 关键影响 → [optional: 一句总结] → 署名
```

**深度调查/特稿：**
```
悬念标题 → 引入场景 → 数据/采访引用 → 具体案例 → 行业分析 → 冷静收尾
```

**年度报告/智库：**
```
报告发布 → 趋势矩阵 → 数据可视化 → 案例详解 → 方法论 → 合作方/团队 → 开放问卷
```
