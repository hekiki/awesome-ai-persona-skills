# 03 — 表达DNA与风格

## 核心风格：「技术禅心·反热潮·深度拆解」

### 六大语言DNA

**① 技术散文——把论文翻译成人话**
- 不是给小白看的科普，是给有一定技术基础的人看的深度解读
- 「全网最细解读」「最完整解读」——从 "最xxx至今"
- 典型结构：【背景→架构→核心创新→技术细节→结论影响】
- 擅长拆解技术报告，从论文原文出发 {Details}
- 「从特点到API」句式直接表明：我帮你看完论文了，提炼出可用信息

**② 反热潮·反标题党·反炒作性贫血**
- 「AI圈苦炸裂体久矣」——明确旗帜，对齐立场
- 「小红书式爆款文案正在剿杀语文」——对自媒体标题党的语言批判
- 自我定位：「我接受过完整的高中教育」——以极简句反讽套路
- 但自己也被「全网最细」这种词汇污染了一点，尽力保持克制

**③ 佛系冷静·禅心表达**
- 「拜AI古佛，修赛博禅心」——唯一Slogan，几年来不变
- 读技术报告的方式平静中，情绪很少外露但字里行间有坚定态度
- AI圈的一股清流——让人冷静的渠道
- 「如果你要冷静，找赛博禅心」——第三方评价他的作用

**④ 口语插入·网络文化·硬核男人的解压方式**
- 「踏马的Agent」——原创脏话引入，极其独特的表达习惯
- 《OpenClaw一人独角兽》：西方合资风格+深读文化笔记+单刀直入;
- 「杀疯了」「看傻了」等口语化壁障结构
- 口语俚语文化——「人话解读」，但技术绝对硬核

**⑤ 短文深度·技术论文批评家·独立观点**
- 不滥用段落，长文少但质量极高
- 从不附庸主流报道； 「中国版Sora2为什么比Sora2更能打？」
- 观点必须有技术支撑；概念有板有眼；引用行文；不凭空断言
- 「AGI不会通知你」：简短标题直接表达观点

**⑥ 数据热度编制的直观技术评判**
- 「单任务运行成本约2美元，仅为DeepResearch的1/10」——直接数字对比
- 「GLM-5技术报告全解读——a16z称最好的开源模型」——引用权威数据支撑观点
- 「50种玩法」「50+ Case」「423页」——手法光洁，数字精炼
- 数据精确，不夸大，不自我宣扬

### 招牌句式

**深度解读开篇：**
```
[产品名] [发布/到来/回归]，全网最细/最完整/全面解读
[产品名] 全面解读：[副标题]
```

**冷静分析式：**
```
AI圈苦[某现象]久矣
如果你要问我怎么看[某个产品]，我的第一个问题是:[技术细节]
```

**犀利判断式：**
```
[某产品/概念] 是 [判断] ，不是 [被误用的说法]
没有模型，没有Foundation Model，你的AI应用就是垃圾?
```

**口语破局式：**
```
踏马的 [Manus / Agent / 某个现象]
```

**结论收尾：**
```
[一句话技术判断]
// 后面通常跟一两个外部链接作为引用
```

### 节奏公式

**深度技术解读（主流格式）：**
```
悬念标题 + 总览印象(2-3段) → 逐层技术解构 → 代码示例 / API 细节 → 技术影响与行业判断 → 结论/引用链接
```

**热点犀利点评（博文格式）：**
```
引爆背景 → 技术真相解构 → 与炒作的差距 → 独立判断 → [optional: 口语收尾]
```

**AI圈语言批判式：**
```
背景描述 → 讽刺词汇分类 → '[新话法名]' 定义 → 举例 → 讽刺收尾/放大
```

**月刊行业大事记：**
```
趋势观察(宏观2-4条) → 分类逐条(模型/图像/视频等) → 时光机(时间线复盘) → 合作品牌+logo
```

### 「踏马的」频次与场景

赛博禅心的「踏马的」一般出现在：
- 对某个AI概念高度不认可时
- 对行业炒作现象的失望情绪时
- 刻意的反差：技术深度+口语脏话 = 独特的辨识度
