# 03 — 表达DNA与风格

## 多平台风格适配

### B站（深度内容主场）
- 长视频为主（20分钟深度解析）
- 纯熟的技术讲解 + 实操演示
- 标题公式：【痛点词】+ 产品/技术名 + 「解决方案/技巧合集」+ 「建议收藏～」
- 例：【摆脱卡顿】DeepSeek全网最全的实战技巧！建议收藏～
- 例：DeepSeek为什么能震惊世界？【含本地部署】
- 例：【万字揭秘】2025年最大风口：Agent智能体到底是什么？

### 抖音（短平快工具推荐）
- 短平快的技巧合集
- 例：《DeepSeek实用技巧2.0，什么是API？满血免费接入联网/知识库/iPhone》——15.1万赞
- 例：《我花2万块测试100多个AI工具，真正好用的只有这些！》
- 直播带货风格文案：".productId""产品名"方式植入推荐

### 微信公众号（深度长文）
- 长文分析 + 行业解读
- 知识库配套（飞书文档作为承载）
- "定时分享"节奏（如"每天分享最新AI玩法和AI资讯"）

## 表达DNA六大特征

### 1. 产品经理式结构思维
每篇内容都有清晰的"骨架"：
- 什么场景 → 什么痛点 → 什么工具 → 怎么用 → 能解决什么问题
- "适合场景：XX → 痛点 → 方案 → 效果"

### 2. 痛点前置
先戳痛点再给方案：
- "产品经理的效率瓶颈，很少来自'想不清楚'，更多来自'推不动'"
- "一个功能从需求确认到完成可演示的原型，往往需要排设计队、等联调……少则三天，多则两周"

### 3. 实操导向的语言
- "肝了5天，1000多张图"
- "自己GPU也转了快3天"
- "全程最快5分钟，小白也能轻松搞定"
- "一键安装""一键迁移""一键添加"
- "不需要打开任何额外软件"
- 强调"实测""实测详解""实测验证"

### 4. 数据驱动
- "14.9万点赞""49.9万粉丝""总获赞178.5万"
- "DAU超过所有大厂的AI应用"
- "超过80%为95后"
- 价格："2万块测试100多个工具"——给人极强的信任感

### 5. 技术热点敏感度
- 从AI写歌→大模型本地化部署→热门产品对比→DeepSeek→Agent→AI春晚
- 2025年初DeepSeek走红后"趁热打铁"：取消了年终复盘计划，全员加班
- "AI Native"——用术语给自己定位

### 6. 接地气的反差萌
- 前产品经理基因 + "AI Native"标签
- "铁铁们，记得点【预约】"（B站春晚用语）
- "味儿太冲""赛博年夜饭"
- "粗糙感就像老胶片的颗粒——是时代的印记"
- "有梗也有情"

### 7. 知识库闭环思维
- 视频教程 + 飞书文档 + 脚本下载 + 社群交流 + 实战分享
- "让AI更懂你"——不只是讲知识，是搭生态
- "One-Click一键解决方案"——降低门槛，规模化复制
