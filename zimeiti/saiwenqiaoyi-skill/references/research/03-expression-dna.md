# 03 — 表达DNA与风格

## 核心风格：「产品老炮的科技速递」

### 六大语言DNA

**① 短视频节奏——1-3分钟高密度输出**
- 不绕弯，开篇3秒给结论/悬念
- 中间快速展示产品/技术亮点
- 结尾一句总结+标签
- 典型结构：「你知道吗？[前沿技术] + [产品/工具展示] + [结论]"——[话题标签]」

**② 产品思维叙事——不是"AI多牛"，而是"产品好在哪"**
- 工业设计师的视角：不只是技术原理，更重要的是"设计得好不好用得好不好"
- 从硬件体验、交互设计、用户体验角度切入AI话题
- "买啥VisionPro，平替足够好了"——用产品思维判断价值

**③ "AI降临派"立场——技术乐观主义但不盲目**
- 不是无脑吹AI，也不是AI恐惧论
- 相信AI将深刻改变世界，但也会客观指出局限性
- "AI降临派®"——把这个标签注册商标化了

**④ 信息平权理念——打破信息差**
- "科技不应该有信息差和壁垒"——这是核心信念
- 第一时间发布、第一时间体验、第一时间分享
- 让普通人也能第一时间接触到前沿科技

**⑤ 短视频标题公式**
- 「你知道吗？」式悬念
- 「还能这么用？」式反差
- 「打工人该享福了」式情绪共鸣
- 「中国版Sora2来了，为啥比Sora2更能打？」式对比冲突

**⑥ 标签体系**
- #玩儿个很新的东西 —— 最具辨识度的个人标签
- #人工智能 #科技改变生活 #AIGC —— 标准标签
- #干货分享 #影视制作 #机器人 —— 场景标签

### 招牌句式

**开篇钩子：**
"你知道吗？[最新科技突破]"
"还能这么用？[AI工具新玩法]"
"第一次尝试用AI[某件事]"

**情绪表达：**
"这哥们发现了[某产品]的终极杀手级应用"
"打工人该享福了"
"开年王炸！"

**结论收尾：**
"[产品]到底智能在哪？"
"未来已来"

### 节奏公式

**科技快讯短视频（主流格式）：**
```
悬念标题 → 3秒钩子 → 产品/技术展示 → 关键亮点(1-2个) → 结论/总结
(全程1-3分钟)
```

**产品体验视频：**
```
标题制造对比 → 实际体验展示 → 优缺点对比 → 结论判断
```

**行业趋势解读：**
```
热点事件 → 深度分析(为什么重要) → 对普通人的影响 → 前瞻预判
```

### 与卡兹克的风格对比

| 维度 | 数字生命卡兹克 | 赛文乔伊 |
|------|---------------|---------|
| 形式 | 公众号长文(3000-8000字) | 短视频(1-3分钟) |
| 情绪 | 阶梯式"卧槽"递进 | 短平快节奏，直接给结论 |
| 数据 | 极度精确(693+185→60) | 有数据但不极端追求精度 |
| 自嘲 | 大量自嘲+幽默 | 较少自嘲，更专业直接 |
| 视角 | AI自媒体人+产品思维 | 工业设计师+产品老炮 |
| 深度 | 万字长文实测 | 短视频快讯+关键信息提炼 |
| 更新频率 | 宁缺勿滥，精品主义 | 高频更新(413期合集体量) |
| 核心能力 | 实测+方法论总结 | 产品洞察+科技敏锐度 |
| 共同点 | 都相信"未来已来"、都关注AI产品实测、都是AI降临派 |
